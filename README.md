# React Tutorial

This uses [the React tutorial](http://facebook.github.io/react/docs/tutorial.html), and incorporated the [MOD's Quote API] (http://dev.markitondemand.com/MODApis/)

## To use

Start the server

```sh
npm install
node server.js
```

And visit <http://localhost:3000/>.
